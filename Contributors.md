# Contributors

This crate has a long history. It was first written by [Walther Chen](https://github.com/hwchen) as an "API library plus credential store" combination. Currently maintained by [Dan Brotsky](https://github.com/brotskydotcom), it is now just a "sample code" crate, with the library/API parts now part of the [keyring-core crate](https://crates.io/crates/keyring-core) and the credential stores all in [separate crates](https://crates.io/search?q=keyring%20credential%20store) of their own.

This file contains a list of known contributors to all generations of this crate, including those whose contributions have now been moved out to other crates. All the contributors listed here are listed by their GitHub monikers, with an `@` in front. Each contributor's name is linked to their GitHub profile page.

If you should be listed in this file but don't find yourself, please contact [@brotskydotcom](https://github.com/brotskydotcom).
